# Pavel Vraj Kodi Add-ons Repository

Tento addon prida do Kodi online repository pro HA Stream Cinema.

Instalace:

1. Stahni ZIP `repository.pavelvraj.kodi-ha-streamcinema-0.1.0.zip` ze slozky `zips/repository.pavelvraj.kodi-ha-streamcinema`.
2. V Kodi zvol `Install from zip file`.
3. Po instalaci repository zvol `Install from repository` a nainstaluj `HA Stream Cinema`.
